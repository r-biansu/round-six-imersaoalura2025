# projeto-imersao-alura-2025-round-six
Projeto inicialmente desenvolvido durante a Imersão Alura 2025 utilizando CodePen e com o tema Round Six. Originalmente pensado apenas para mostrar resultados no console, mas recentemente iniciei o processo de apresentar as funções na página e não no console. Sendo o "Conversor de Moedas" e o "Quiz" o ponto inicial desse aprimoramento.
